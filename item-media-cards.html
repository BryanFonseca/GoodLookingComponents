1 #Media Cards Media Cards

Toggle example guides Toggle HTML markup

Shows a cards stacked one after another (side-by-side) controlling x-overflow with scrolling. This module is intended to be used with the Card module.

Use the following JavaScript to set the z-index property on every Card component so that they don't overlap.

  const cards = document.querySelectorAll('.card');
  let depth = 0;
  cards.forEach((card) => {
    card.style.zIndex = depth;
    depth++;
  });
